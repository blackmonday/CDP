Add child organisations

This section allows you to add child organisations to your organisation's profile. Once added, child organisations will appear below. You can add new child organisations or remove them at any time.

  • only a parent organisation can add its child organisations
  • a child organisation can only be added if they are registered in the Supplier Registration service (opens in a new tab)
What is a child organisation?

Public bodies are organised within a hierarchical structure that connects organisations. This makes it easier to report on spending and contracts. A parent organisation sponsors one or more child organisations, which report directly to them. Only a parent can add its child organisations.
